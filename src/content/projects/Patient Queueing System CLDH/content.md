## Overview

Built a real-time queue system integrated with BizBox HIS to manage patient flow and reporting. The system features a NextJS frontend for real-time queue management, a WinForms application for patient registration, and a backend API that interfaces with the BizBox HIS database using Dapper for efficient data access. Socket.IO is used to enable real-time updates across the system, ensuring that all components remain synchronized. This project significantly improved patient flow management and reporting accuracy within the healthcare facility.
